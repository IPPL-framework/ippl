Detailed Description

Particle-tile scatter: bin -> per-team scratch histogram -> flush.

Each team owns one tile of the output grid; particles assigned to that tile are walked once, scattering into LDS scratch, which is then flushed back to global memory. Drastically reduces atomics traffic at high particles-per-cell on GPU.
