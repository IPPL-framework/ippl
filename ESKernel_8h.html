Detailed Description

Exponential-of-Semicircle (ES) spreading kernel for NUFFT.

The header exposes:

  • es_kernel_eval_wN<T> device-friendly polynomial evaluators for each precomputed kernel half-width N (3..15).
  • ESKernel<T> a runtime selector that picks the smallest width achieving the requested tolerance and dispatches to the matching polynomial.

Coefficients are minimax polynomials in t=x^2 with magic literals encoded as hex floating-point so they are exact across compilers.
