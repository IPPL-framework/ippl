Detailed Description

template<typename T, unsigned Dim, typename MemSpace, typename Tag>
class ippl::fft::HeffteTrig< T, Dim, MemSpace, Tag >

Wrapper around heffte::fft3d for sine / cosine transforms.

Specializations are generated by IPPL_FFT_DEFINE_HEFFTE_TRIG for each transform tag (SineTransform, CosTransform, Cos1Transform). The tag selects the corresponding heFFTe backend.

Template Parameters
TReal precision type.
DimSpatial dimension.
MemSpaceKokkos memory space holding the buffers.
TagOne of SineTransform, CosTransform, Cos1Transform.
