Detailed Description

template<int D>
class ippl::detail::GPModel< D >

ARD Gaussian-process regressor with gradient-ascent hyperparameter fit.

Used by the auto-tune sweep for low-dim Bayesian optimization over tile / team / oversubscription configurations.

Template Parameters
DNumber of input dimensions (small: 1..10).
