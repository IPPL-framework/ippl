Detailed Description

template<typename RealField>
class ippl::FFT< NUFFTransform, RealField >

Non-uniform FFT for IPPL particles + uniform Fourier modes.

Supports both the native IPPL implementation and the FINUFFT (CPU / GPU) backend. The runtime backend choice is controlled by the "useFinufft" / "lockMethod" parameter keys plus the configure macros ENABLE_FINUFFT and ENABLE_GPU_NUFFT.

Type 1: non-uniform points -> uniform grid (spreading / adjoint). Type 2: uniform grid -> non-uniform points (interpolation).

Template Parameters
RealFieldIPPL Field of real values used for grid sizing.
