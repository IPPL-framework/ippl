Detailed Description

template<typename Kernel, unsigned Dim>
class ippl::Gather< Kernel, Dim >

Public functor that interpolates a Field at particle positions.

Constructed with a kernel and an optional GatherConfig. The call operator invokes the configured backend (Atomic / AtomicSort) via WidthDispatcher so the kernel width is known at compile time inside the inner loop.

Template Parameters
KernelInterpolation kernel type.
DimSpatial dimension.
