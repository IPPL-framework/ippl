Detailed Description

template<typename Field, typename OperatorF>
struct ippl::multigrid_preconditioner< Field, OperatorF >

Multigrid preconditioner for linear solvers.

This preconditioner uses a V-cycle multigrid approach to accelerate the convergence of iterative solvers like PCG.

Template Parameters
FieldThe type of field used for the solution and source.
OperatorFThe type of the operator (e.g., a Laplacian).
