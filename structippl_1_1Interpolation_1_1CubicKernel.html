Detailed Description

template<typename T = double>
struct ippl::Interpolation::CubicKernel< T >

Cubic B-spline kernel (PCS / Piecewise Cubic Spline / order 3).

Natural form (y in [-2, 2]): phi(y) = 2/3 - y^2 + |y|^3/2 for |y| < 1 = (2 - |y|)^3 / 6 for 1 <= |y| < 2 = 0 otherwise

With inv_hw = 2/w = 0.5, input x relates to natural y by: y = x * 2 So kernel receives x in [-1, 1], transforms to y in [-2, 2]
