Detailed Description

Multi-node cuFFTMp wrappers for IPPL FFT transforms.

Provides C2C, R2C and pruned variants on top of cuFFTMp. Includes small CUDA helpers for layout transposes (LayoutLeft <-> LayoutRight) since cuFFTMp expects row-major data while IPPL uses Kokkos LayoutLeft views.
