Detailed Description

Non-uniform FFT specialization (NUFFTransform tag).

Provides a single FFT class that can dispatch to the native IPPL NUFFT (Kokkos-based ES kernel + heFFTe) or to FINUFFT / cuFINUFFT when those libraries are enabled at configure time.
