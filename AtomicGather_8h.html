Detailed Description

Per-particle gather kernel.

Each particle reads its W^Dim stencil from the field. The "Atomic" name is kept for symmetry with the scatter side; gather is read-only and never actually issues atomic operations. With UseSorting=true a permutation is applied so particles in the same tile read contiguous memory.
