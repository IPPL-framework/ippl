Detailed Description

template<typename T = double>
struct ippl::Interpolation::LinearKernel< T >

Linear kernel (CIC / Cloud-in-Cell / order 1).

Natural form: phi(y) = 1 - |y| for |y| < 1, else 0 With inv_hw = 2/w = 1, input x = y, no transformation needed Normalized: phi(x) = 1 - |x| for |x| < 1
