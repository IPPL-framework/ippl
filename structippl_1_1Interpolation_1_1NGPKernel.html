Detailed Description

template<typename T = double>
struct ippl::Interpolation::NGPKernel< T >

Nearest Grid Point kernel (NGP / order 0).

Natural form: phi(y) = 1 for |y| < 0.5, else 0 With inv_hw = 2/w = 2, input x = y * 2, so y = x * 0.5 Normalized: phi(x) = 1 for |x| < 1
