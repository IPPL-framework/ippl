enum class ippl::Interpolation::GatherMethod
strong

Gather method for grid-to-particle interpolation.

  • Atomic: read-only field gather, no sorting (no atomic ops actually needed since gather only reads – name kept for symmetry with the Scatter API).
  • AtomicSort: same gather kernel, with a binning pre-pass that improves cache locality for clustered particle distributions.

◆ ScatterMethod

enum class ippl::Interpolation::ScatterMethod
strong

Scatter algorithm for particle -> grid interpolation.

  • Atomic: per-particle stencil scatter with Kokkos::atomic_add.
  • Tiled: bin particles into tiles, scatter each tile's histogram into team-local scratch, then flush.
  • OutputFocused: same as Tiled but with grid-parallel histogram filling (one team per output tile, batched particle reads).
