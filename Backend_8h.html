Detailed Description

Aggregate include for all FFT backends supported by IPPL.

Pulls in the heFFTe wrappers unconditionally, plus optional CUDA-only (CuFFT) and CUDA+MPI (CuFFTMp) backends gated by the corresponding Kokkos / IPPL configuration macros.
