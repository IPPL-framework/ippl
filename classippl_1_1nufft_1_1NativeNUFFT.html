Detailed Description

template<unsigned Dim, typename T = double, typename ExecSpace = Kokkos::DefaultExecutionSpace>
class ippl::nufft::NativeNUFFT< Dim, T, ExecSpace >

NUFFT implementation.

Type 1: Spread from nonuniform points to uniform Fourier modes (scatter -> FFT -> deconvolution)

Type 2: Interpolate uniform Fourier modes at nonuniform points (correction -> FFT -> gather)

Template Parameters
DimNumber of dimensions
TFloating point type
ExecSpaceKokkos execution space

Type 1 NUFFT: Spread from nonuniform points to uniform Fourier modes.

Computes f_k = sum_j c_j * exp(i * k * x_j) for uniform k.

Template Parameters

Type 2 NUFFT: Interpolate uniform Fourier modes at non-uniform points.

Computes c_j = sum_k f_k * exp(-i k x_j) for non-uniform x_j. Pipeline: pre-correction -> inverse FFT (or pruned FFT) -> gather.

Template Parameters
InFieldField type holding the input Fourier modes.
Parameters
fInput Fourier-mode field.
RParticle positions in [0, 2*pi)^Dim.
QOutput particle values.
upsampled_outputUse the full upsampled grid (true) or the pruned-FFT path (false; required on distributed runs).
