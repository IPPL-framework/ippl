Detailed Description

template<typename RealField>
class ippl::FFT< PrunedRCTransform, RealField >

Pruned R2C FFT keeping only the lowest n_modes per axis.

Currently 3D-only. Forward maps a real field to a pruned complex field; backward goes the other way. Internally uses a full-size complex scratch view so the index arithmetic stays local.
