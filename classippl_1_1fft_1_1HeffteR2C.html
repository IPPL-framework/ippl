Detailed Description

template<typename T, unsigned Dim, typename MemSpace>
class ippl::fft::HeffteR2C< T, Dim, MemSpace >

Wrapper around heffte::fft3d_r2c for real-to-complex transforms.

forward() consumes a real buffer and writes the half-complex spectrum; backward() does the inverse. Normalization matches HeffteC2C: forward is fully normalized, backward is unscaled.

Template Parameters
TReal precision type.
DimSpatial dimension.
MemSpaceKokkos memory space holding the buffers.
