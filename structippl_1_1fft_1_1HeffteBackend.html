Detailed Description

template<typename MemSpace>
struct ippl::fft::HeffteBackend< MemSpace >

Pick the appropriate heFFTe backend types for a memory space.

Provides nested aliases c2c, sin, cos, cos1 for the complex / sine / cosine / cosine-Type-I transforms. Specializations select FFTW / MKL on the host, cuFFT on CUDA, rocFFT on HIP, and fall back to the heFFTe stock backend everywhere else.
