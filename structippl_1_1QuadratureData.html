Detailed Description

template<typename TVal, typename TDeriv, unsigned numElementDOFs>
struct ippl::QuadratureData< TVal, TDeriv, numElementDOFs >

Per-quadrature-node basis data passed to evaluateAx evaluator functors.

Template Parameters
TValType of basis values at each local DOF (scalar T for Lagrange, Vector<T, Dim> for Nedelec).
TDerivType of the spatial derivative data at each local DOF (gradient for Lagrange, curl for Nedelec).
numElementDOFsNumber of local DOFs per element.
