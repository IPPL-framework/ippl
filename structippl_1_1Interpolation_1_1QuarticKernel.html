Detailed Description

template<typename T = double>
struct ippl::Interpolation::QuarticKernel< T >

Quartic B-spline kernel (order 4).

Natural form (y in [-2.5, 2.5]): phi(y) = (115/192) - (5/8)y^2 + (1/4)y^4 for |y| < 0.5 = (55/96) + (5/24)|y| - (5/4)y^2 + (5/6)|y|^3 - (1/6)y^4 for 0.5 <= |y| < 1.5 = (5/2 - |y|)^4 / 24 for 1.5 <= |y| < 2.5 = 0 otherwise

With inv_hw = 2/w = 0.4, input x relates to natural y by: y = x * 2.5
