Detailed Description

template<typename T, unsigned Dim, typename MemSpace>
class ippl::fft::CuFFTMpR2C< T, Dim, MemSpace >

Distributed real-to-complex FFT via cuFFTMp.

Holds two cuFFTMp plans (R2C and C2R) plus a CUDA stream. Mirrors the IPPL R2C backend interface. Forward = real -> half-complex (normalized by 1/N); backward = half-complex -> real (unscaled).

Template Parameters
TReal precision (float / double).
DimSpatial dimension (only 3D).
MemSpaceKokkos memory space.
