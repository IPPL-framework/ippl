Detailed Description

template<typename T, unsigned Dim, typename MemSpace>
class ippl::fft::HeffteC2C< T, Dim, MemSpace >

Thin wrapper around heffte::fft3d for complex-to-complex transforms.

Owns the heFFTe plan and a workspace large enough to handle the configured maximum batch size. forward() applies full normalization (heffte::scale::full); backward() applies none, so a forward followed by a backward returns the input.

Template Parameters
TReal precision type (float / double).
DimSpatial dimension (only 2 and 3 are supported).
MemSpaceKokkos memory space holding the input/output buffers.
